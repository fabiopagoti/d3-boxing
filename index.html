D3 Boxing

How to play:
Boxer 1 Use keys W, A, S, D to move ans space to punch
Boxer 2 Use arrow keys to move ans enter to punch

Note: The game on this site is the original version of D3 Boxing.Other versions are derivatives or fakes, and should be used with caution.

Created by Fabio Pagoti Based on Atari 2600 Boxing created by Bob Whitehead (Activision) which is the first video game I remember playing.
